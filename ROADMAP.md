# Roadmap

## v0.2

- add `setup.sh` for a one-command public-safe demo
- add local X home JSON processing pipeline
- add `GETTING_STARTED.md` with privacy-first onboarding
- add `examples/` with before/after sanitized outputs

## v0.3

- add a private live X connector template that can run with a dedicated browser profile
- add a more complete X dashboard with feed health, learning queue, and reflection panels
- add optional Ollama and LM Studio distillation backends for local summarization and reflection
- add a clearer handoff between private capture and public learning layers

## v0.4

- add agent write-back templates for promoting new rules into memory
- add multi-source input beyond X, including reading queues and local notes
- add promotion policies for moving transient signals into durable memory
- add lightweight tests for config resolution and learning pipeline outputs

## Later

- support multiple private memory backends behind the same public framework
- add richer local review surfaces beyond a static HTML dashboard
- document repeatable second-brain operating patterns for different user types

## Non-Goals

- shipping anyone's personal memory
- bundling production account sessions
- providing a hosted SaaS product
- optimizing for large-scale multi-user infrastructure first
