# Roadmap

## Near Term

- improve the starter onboarding flow
- add one mock ingestion workflow end to end
- expand the demo dashboard to show more memory-layer state
- add lightweight tests for config and memory sync behavior

## Mid Term

- add public-safe examples for concept pages, entity pages, and source notes
- add optional connector interfaces for social feeds, reading queues, and local files
- add promotion rules from transient notes into durable memory
- define a clearer schema for agent writeback

## Long Term

- support multiple private memory backends behind the same public framework
- add richer local review surfaces beyond a static HTML dashboard
- make reflection, distillation, and memory promotion easier to customize
- document repeatable second-brain operating patterns for different user types

## Non-Goals

- shipping anyone's personal memory
- bundling production account sessions
- providing a hosted SaaS product
- optimizing for large-scale multi-user infrastructure first
